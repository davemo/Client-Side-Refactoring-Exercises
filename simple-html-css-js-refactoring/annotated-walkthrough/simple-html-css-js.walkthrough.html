The first thing we'll do is eliminate the div-itis going on and plug in a more semantic ol tag with li for children. 

    <ol id="refactoring-steps"> <!-- this id is much more semantically meaningful. -->
¶

Replacing divs with li elements allows us to remove the text-indent: 10px; that was previously there. 

      <li>Refactor <em>inline styles</em> (into the head tag or an external stylesheet).</li> <!-- we swapped out the div#highlightedBlackText with a simple usage of the em element. -->
      <li>Refactor <em>duplication</em> that exists in the inline style attributes.</li>
      <li>Refactor <em>markup</em> to use appropriate html elements.</li>
      <li>Refactor <em>styles</em> to eliminate the property duplication.</li>
      <li>Refactor <em>javascript</em> to eliminate inline click handler.</li>
    </ol>
¶

Then, we can migrate the inline styles to a style block. 

    <style type="text/css">
      #refactoring-steps { /** adding an id for the ordered list allows us to target it and apply the default state of hidden **/
        visibility: hidden;
      }
      
      ol em { /** removing the divs tha were used for marking up emphasis and replacing with em tags allows us to eliminate the display: inline; as em tags already display inline due to browser defaults **/
        font-weight: bold;
        color: blue;
      }
    </style>
